# 📘 Design and Analysis of Algorithms (DAA) – C++ Programs

This repository contains **C++ implementations of Design and Analysis of Algorithms (DAA)** problems, organized **week-wise** as per academic coursework. Each week focuses on a specific set of algorithmic concepts, progressing from basic searching techniques to advanced divide-and-conquer and selection algorithms.

The goal of this repository is to provide:

* Clear and structured **week-wise practice**
* Easy **revision before exams**
* Readable and testable **C++ implementations**

## 🧠 Weekly Breakdown

### ✅ Week 1 – Searching Algorithms

* Linear Search (Iterative)
* Linear Search (Recursive)
* Binary Search
* Jump Search

**Concepts Covered:**

* Basic searching techniques
* Time complexity comparison
* Recursive vs iterative approaches

---

### ✅ Week 2 – Array & Binary Search Applications

* Count occurrences of an element using Binary Search
* Find triplets with a given sum
* Count pairs with a given difference

**Concepts Covered:**

* Modified binary search
* Problem-solving on sorted arrays
* Optimized brute-force alternatives

---

### ✅ Week 3 – Sorting Algorithms with Analysis

* Insertion Sort with comparison count
* Selection Sort with comparison count

**Concepts Covered:**

* Elementary sorting algorithms
* Performance measurement using comparisons
* Best, average, and worst-case analysis

---

### ✅ Week 4 – Divide and Conquer Algorithms

* Merge Sort with inversion count
* Randomized Quick Sort
* Kth smallest element using Median of Medians

**Concepts Covered:**

* Divide and conquer paradigm
* Randomization in algorithms
* Deterministic linear-time selection

## 🛠️ How to Compile and Run

Use any standard C++ compiler (GCC recommended):

```bash
g++ filename.cpp -o output
del output.exe
```

Example:

```bash
g++ W1Q2_BinarySearch.cpp -o binarySearch
binarySearch
```
